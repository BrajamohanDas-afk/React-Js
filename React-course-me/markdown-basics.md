# Markdown Basics Cheat Sheet

## Headings

**Code:**
```
# Heading 1
## Heading 2
### Heading 3
```

**Output:**
# Heading 1
## Heading 2
### Heading 3

---

## Text Formatting

**Code:**
```
**Bold Text**
*Italic Text*
***Bold and Italic***
~~Strikethrough~~
`Inline Code`
```

**Output:**

**Bold Text**

*Italic Text*

***Bold and Italic***

~~Strikethrough~~

`Inline Code`

---

## Lists

### Unordered List

**Code:**
```
- Item 1
- Item 2
  - Nested Item 2.1
  - Nested Item 2.2
- Item 3
```

**Output:**
- Item 1
- Item 2
  - Nested Item 2.1
  - Nested Item 2.2
- Item 3

### Ordered List

**Code:**
```
1. First Item
2. Second Item
3. Third Item
```

**Output:**
1. First Item
2. Second Item
3. Third Item

---

## Links

**Code:**
```
[Click Here](https://www.google.com)
```

**Output:**

[Click Here](https://www.google.com)

---

## Images

**Code:**
```

```

**Output:**

## Blockquotes

**Code:**
```

## Tables

**Code:**
```
| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
```

**Output:**

| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |

---

## Task Lists

**Code:**
```
- [x] Completed task
- [ ] Incomplete task
- [ ] Another task
```

**Output:**
- [x] Completed task
- [ ] Incomplete task
- [ ] Another task

---

## Line Breaks

**Code:**

```

**Output:**
